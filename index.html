1. Difference between flex row and column?

In CSS, the flex boxes was designed as one-dimensional layout, means we can deal with items either row or column at once.We use flex-direction: row when we use for rows and we use flex-direction:column when we use for column.

3. Justify-content allows you to do what?

In CSS, Justify-content allow us to identify how the browser is distributed the space between and space around to an element of the flex container as well as grid container.

4. Difference between absolute and relative positioning?

In CSS, the element is Positioned absolutely to its first positioned parent. Whereas the element is positioned relative to its normal position.

5. What is grid-template-columns used for?

In CSS,the grid-template-columns property specifies the number (and the widths) of columns in a grid layout. The values are a space separated list, where each value specifies the size of the respective column

6. What is the z-index in CSS?

In CSS,the z-index property specifies the stack order of an element. An element with greater stack order is always in front of an element with a lower stack order.

7. What is the difference between padding and margin?

In CSS, a margin is the space around an element's border, while padding is the space between an element's border and the element's content.
